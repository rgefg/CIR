# LLM Appendix Examples

This folder contains five cleaner LLM-generated supervision examples selected from `data/cc3m_cir_dataset_cleaned_v1mid_v2_with_reverse.jsonl` for appendix use.

Files:
- `LLM_APPENDIX_EXAMPLES.md`: appendix-ready summary table.
- `llm_appendix_examples.jsonl`: exact selected records with category and source-evidence notes.
- `llm_appendix_examples.csv`: spreadsheet-friendly version.
- `llm_appendix_examples.tex`: compact LaTeX table body.
